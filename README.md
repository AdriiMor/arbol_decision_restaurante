# Sistema de Recomendación de Platillos con Árbol de Decisión

Proyecto desarrollado en Go que utiliza un árbol de decisión para recomendar platillos y bebidas de acuerdo con las preferencias del usuario, considerando factores como gusto por la comida picante, consumo de carne, preferencia por alimentos dulces y selección de bebidas.

## Características
- Implementado en Go.
- Uso de árboles de decisión para la recomendación.
- Selección personalizada de platillos.
- Recomendación de bebidas según el platillo elegido.
- Interfaz web desarrollada con HTML y CSS.

## Archivos del proyecto
- main.go: Código principal.
- index.html: Interfaz de usuario.
- styles.css: Estilos de la aplicación.
- go.mod: Dependencias del proyecto.

## Ejecución

```bash
go run main.go
```
